OBSCURA

Panduan Penggunaan

Selamat datang di Obscura. Ini adalah proyek independen yang lahir dari sebuah hobi dan hasrat untuk dunia sulap, terutama seni mentalisme. Dikonsepkan oleh @pesulapgabut, aplikasi ini bukanlah produk komersial dan tidak untuk diperjualbelikan.

Obscura didedikasikan sepenuhnya untuk dapat digunakan secara gratis oleh teman-teman terdekat sebagai sarana untuk menyalurkan ide dan kreativitas. Selamat menggunakan.

1. Memulai & Terhubung

  • Membuat ID Anda:
    Setiap pengguna harus memiliki ID 6 digit. Buka menu Settings, lalu masukkan 6 digit angka di bagian "ID Anda" untuk membuat identitas permanen Anda.
  • Menghubungkan dengan Partner:
    Di menu Settings, masukkan ID teman Anda di kolom "Partner", lalu tekan "Kirim Permintaan Koneksi". Teman Anda harus menekan "Izinkan" pada notifikasi yang muncul di halaman utamanya.

2. Halaman Sender (Pengirim)

Halaman ini berfungsi sebagai "remote control" Anda untuk mengirim informasi. Anda hanya bisa mengaksesnya saat terhubung dengan seorang Partner.

  • Input Teks

    Ketik pesan atau kata apa pun, lalu tekan Enter pada keyboard Anda untuk mengirimkannya.

  • Input Angka Cepat

    Gunakan tombol angka (0-9) untuk mengirim satu digit angka secara instan dengan sekali tekan.

  • Kalkulator

    Gunakan kalkulator untuk melakukan perhitungan. Saat Anda menekan tombol =, hasilnya akan langsung dikirim ke partner Anda dan display kalkulator akan otomatis bersih, siap untuk perhitungan berikutnya.

3. Halaman Receiver (Penerima)

Halaman ini berfungsi sebagai "layar" Anda untuk menerima informasi dari Sender atau Tamu.

  • Penerimaan Real-time

    Layar akan secara otomatis menampilkan data apa pun (teks, angka, atau hasil kalkulator) yang dikirimkan kepada Anda. Data yang masuk bersifat sesaat dan akan digantikan oleh data baru.

  • Text-to-Speech (TTS)

    Secara default, suara akan aktif dan membacakan data yang masuk dalam Bahasa Indonesia. Anda bisa menekan tombol ikon speaker di bawah teks untuk mengontrolnya:
    - Tekan sekali untuk mematikan suara (Anda akan mendengar notifikasi "Voice Off").
    - Tekan lagi untuk mengaktifkannya kembali (Anda akan mendengar notifikasi "Voice On").

  • Tombol Clear Display

    Gunakan tombol ini di pojok kanan atas untuk membersihkan layar secara manual jika diperlukan.

4. Fitur Lanjutan: Link Tamu

Fitur ini memungkinkan Anda menerima data dari seseorang yang tidak memiliki aplikasi ini.

  • Di Menu Utama, tekan tombol "Buat Link Tamu".
  • Sebuah kotak dialog akan muncul. Tekan "Salin" atau "Bagikan" untuk memberikan link unik tersebut kepada tamu Anda.
  • Link ini hanya bisa digunakan SATU KALI. Saat tamu terhubung, status di Menu Utama Anda akan menjadi "Tamu Terhubung".
